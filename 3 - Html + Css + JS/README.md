# Flip & Find Game

A fun and interactive memory-matching game built with HTML, CSS, and JavaScript. Test your memory skills by flipping cards and finding matching pairs of adorable animal emojis!

##  Game Overview

Flip & Find is a classic memory card game where players flip cards to reveal animal emojis and match pairs. The goal is to remember the positions of the cards and match all pairs in the fewest moves possible.

##  Features

- **38 Interactive Cards**: 19 unique pairs of animal emoji cards
- **Animal Emojis**: Adorable animals like zebras, giraffes, koalas, penguins, and more
- **Start Game Button**: Begin your game with a single click
- **Restart Game**: Reset the game board anytime with the restart button
- **Shuffle Cards**: Randomize card positions during gameplay
- **Responsive Design**: Smooth card flip animations with CSS transitions
- **Victory Message**: Celebratory congratulations screen when all pairs are matched
- **Sound Effects**: Click sounds for interactive feedback

##  How to Play

1. Click the **"Start Game"** button to begin
2. Click on cards to flip them and reveal the animal emojis
3. Try to find matching pairs by remembering card positions
4. Match all 19 pairs to win the game
5. Use the **"Shuffle"** button to randomize positions if needed
6. Click **"Restart"** to play again

##  Technologies Used

- **HTML5**: Game structure and layout
- **CSS3**: Card animations and styling with 3D flip effects
- **JavaScript**: Game logic, card matching, and event handling

##  Game Rules

- There are 38 cards total (19 matching pairs)
- Each card contains a unique animal emoji
- Click cards to flip and reveal the animal emoji
- Remember the positions of matching animals
- Match all pairs to win
- The game prevents clicking multiple cards simultaneously to ensure fair play
- Successfully matched cards remain flipped
- Sound effects play on each card click

##  Getting Started

1. Open index.html in your web browser or visit https://flip-and-find.netlify.app/
2. Click "Start Game" to begin
3. Start flipping cards and finding matching animal pairs!

##  Game Features in Detail

### Card Flip Animation
- Smooth 3D flip effect using CSS transforms
- Cards have a front face ("Click to Flip") and back face (animal emoji)

### Game Controls
- **Start Game**: Initializes the game board
- **Restart**: Clears the board and starts a new game
- **Shuffle**: Randomizes the card positions without resetting matched pairs

### Victory Condition
- Complete all matches to see the congratulations message
- Play again button available on victory screen

##  Notes

- Animal emojis are dynamically displayed on card flip
- Card positions are randomized each game for variety
- Sound effects provide audio feedback on card clicks
- Prevents rapid clicking to avoid confusion during gameplay
- Perfect for all ages!
